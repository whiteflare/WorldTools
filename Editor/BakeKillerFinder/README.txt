# これは何ですか？

VRChatワールド製作における補助ツールです。
エラーメッシュなどを検索してリストアップします。

もともとバーチャルマーケット３の開発中に発生した、ライトベイク時のUnityクラッシュを解析するためのスクリプトでした。
バーチャルマーケット４の開発においてさらに機能追加(魔改造ともいう)をしたものがこのツールです。

スクリプトの開発は「車軸製作所」さん、魔改造は「フレア」が担当しました。


# 何ができますか？

Missing な Script, Prefab, Material, Mesh などを検索できます。
詳しくは使って憶えてください。
