------------------------------
LightProbe Edit Utility
------------------------------

# これは何ですか？

ライトプローブの可視化・編集を行うエディタ拡張です。

# 使い方

## 可視化

LightProbeVisualizer.prefab をシーン内の任意の場所にロードしてください。
操作は Inspector から行うことができます。また『ツールウィンドウを開く』にて独立した操作パネルを使用することもできます。

## 編集

Hierarchy 右クリックからの『ライトプローブ編集』から操作できます。
一部メニューは複数 LightProbeGroup を選択する必要があるためご注意ください。
