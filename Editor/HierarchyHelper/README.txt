------------------------------
Hierarchy Helper
------------------------------

# これは何ですか？

Hierarchy に対し、特定条件の GameObject の網掛けを行うエディタ拡張です。

# 何ができますか？

EditorOnly や Static 設定など、特定条件の GameObject を網掛けすることで可視化します。
SkinnedMeshRenderer が参照しているボーンの GameObject を網掛けすることもできます。
網掛けしたオブジェクトを選択することもできます。
